# Flutter-Counter

Простое iOS-приложение на Flutter с одной кнопкой и областью вывода текста.
По нажатию:
1. Увеличивается счётчик N;
2. Вызывается Swift-код через MethodChannel;
3. Swift дергает минимальную библиотеку на C++;
4. Библиотека записывает строку hello world N в текстовый файл;
5. Обновлённое содержимое файла отображается в UI Flutter.
